# Knovis prototype

Исполнимый прототип графа организационных знаний для проверки сценария:

`роль -> задачи -> требования -> состояние сотрудника -> дефициты -> готовность -> минимальное действие развития`.

## Запуск

Рекомендуемый вариант на Windows: дважды нажать `open-knovis.cmd`.

Он запустит локальный сервер и откроет прототип в браузере.

Самый простой вариант: открыть `index.html` в браузере.

Локальный сервер:

```powershell
node server.js
```

После запуска открыть:

```text
http://127.0.0.1:5173
```

## Что уже есть

- правая навигация с разделами `Люди` и `Роли`;
- режим роли: выбор роли показывает только граф знаний роли;
- режим человека: выбор человека открывает выбор роли и затем показывает оценку готовности;
- паспорт роли;
- граф связей роли, задач, требований, рисков и действий;
- оценка требований по сотруднику;
- расчет готовности;
- список дефицитов;
- минимально достаточные интервенции для закрытия дефицитов.
